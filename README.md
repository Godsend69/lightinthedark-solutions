A complete 7-page life advisory website built for GitHub Pages deployment.

## Pages
- `index.html` — Home
- `about.html` — About Cyrus Nash, Philosophy, L.I.F.E. Project
- `services.html` — All 5 services (Soul Mapping, Sovereign Path, Crisis Light, Life Architecture, Group Circle)
- `pricing.html` — Monthly / Annual / Lifetime plans with Stripe integration
- `blog.html` — 6 essay cards with category filter
- `resources.html` — Free guides, newsletter signup, recommended reading
- `contact.html` — Full contact form, FAQ, emergency booking

## Before Going Live — 3 Quick Steps

### 1. Stripe Payments
Open `pricing.html` and replace all `YOUR_STRIPE_PAYMENT_LINK_*` values with your real Stripe Payment Links from stripe.com → Payment Links.

### 2. Contact Form
Open `contact.html` and `resources.html`, replace `YOUR_FORM_ID` with your Formspree ID from formspree.io (free account).

### 3. Deploy to GitHub Pages
- Push all files to a new GitHub repository
- Go to Settings → Pages → Source: main branch / root folder
- Your site goes live at: `https://godsend69.github.io/[repo-name]/`
